1) What is the difference between Map,Filter,forEach and find

The difference between map,filter,forEach,find is:
Map: returns a new array that is the result of calling a provided function on each element of the original array.
Filter: returns a new array that includes only the elements from the original array that satisfy a provided condition.
forEach: calls a provided function once for each element in an array, but it does not return anything.
find: returns the first element in an array that satisfies a provided condition. In summary, map and filter both return new arrays, forEach performs some side-effect on each element, and find returns a single element or undefined.

2)What is the difference between localStorage,sessionStorage and cookies

Local storage is useful for storing data that the user will need to access later, such as offline data. Session storage is a great way to improve the performance of your web applications. Cookies are a good choice for storing data that should not be persisted for a long time, such as session IDs.

3) What is event loop

Event loop is manages the execution of multiple asynchronous tasks in a single thread of execution. The event loop works by continuously monitoring a queue of events or tasks and processing them in the order they are received. Each event or task is associated with a callback function that is executed when the event or task is processed.

How does the quizTimer function work

The quizTimer function is used to create a timer that counts the time elapsed for a quiz. It takes a parameter dismiss, which when set to true, clears the timer and stops the countdown. The function uses setInterval to run a callback function every 1000 milliseconds (1 second), which updates the timer display by calculating the minutes and seconds elapsed and displaying them on the HTML page.
